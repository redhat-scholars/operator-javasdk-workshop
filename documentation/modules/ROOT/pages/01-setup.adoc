== Clone Git Repository

`git clone https://github.com/sebastienblanc/greeting-operator.git`

== Connect to a Kubernetes cluster

Ask to your instructor for instruction, if you are using ROKS you can get your cluster https://get-devcluster-stage.apps.sandbox-stage.gb17.p1.openshiftapps.com/[here] 

This workshop also works with minikube

NOTE: This workshop does not work with the OpenShift Developer sandbox

== IDE 

You can use any IDE you want, as long you have support for JDK11 and Maven 

An other option is to use "Red Hat OpenShift Devspaces" you can get access to this by creating a developer https://developers.redhat.com/developer-sandbox[sandbox account]
